Pré-processar Colunas de Atributo¶

Como voc pode ver, h muitas colunas no numricas que precisam ser convertidas! Muitas delas so simplesmente yes/no, por exemplo, a coluna internet. razovel converter essas variveis em valores (binrios) 1/0.

Outras colunas, como Mjob e Fjob, tm mais do que dois valores e so conhecidas como variveis categricas. A maneira recomendada de lidar com esse tipo de coluna criar uma quantidade de colunas proporcional aos possveis valores (por exemplo, Fjob_teacher, Fjob_other, Fjob_services, etc), e assinalar 1 para um deles e 0 para todos os outros.

Essas colunas geradas so por vezes chamadas de variveis postias (em ingls: dummy variables), e ns iremos utilizar a funo pandas.get_dummies() para fazer essa converso. Execute a clula de cdigo abaixo para executar a rotina de pr-processamento discutida nesta seo.

def preprocess_features(X):
    ''' Pr-processa os dados dos estudantes e converte as variveis binrias no numricas em
        variveis binrias (0/1). Converte variveis categricas em variveis postias. '''
    
    # Inicialize nova sada DataFrame
    output = pd.DataFrame(index = X.index)

    # Observe os dados em cada coluna de atributos 
    for col, col_data in X.iteritems():
        
        # Se o tipo de dado for no numrico, substitua todos os valores yes/no por 1/0
        if col_data.dtype == object:
            col_data = col_data.replace(['yes', 'no'], [1, 0])

        # Se o tipo de dado for categrico, converta-o para uma varivel dummy
        if col_data.dtype == object:
            # Example: 'school' => 'school_GP' and 'school_MS'
            col_data = pd.get_dummies(col_data, prefix = col)  
        
        # Rena as colunas revisadas
        output = output.join(col_data)
    
    return output

X_all = preprocess_features(X_all)

1 - Maquina de vetores de suporte¶

SVM (Maquina Vetor de Suporte) um dos algoritimos de classifio que tem por objetivo receber um conjunto de features de entrada e produzir um classificador, em outras palavas encontra o limite entre duas classes com a maior margem de separao possvel.

Aplicação:¶

SVM utilizado para aplicao na avaliao da solvencia de cooperativas de crditos.

fonte: (https://repositorio.ufsc.br/xmlui/bitstream/handle/123456789/158361/318899.pdf?sequence=1&isAllowed=y)

Vantagens¶
  • Tem melhor desempenho em domnios complicados em que existe uma clara separao entre os classificadores. -Grande capacidade de generalizao, para medir tal capacidade basta utilizar os classificadores no conjunto de teste que no foi usado no treinamento. -Lida bem com outlier, a ponto de desconsidera-lo para melhorar a classificao.ue h presena de mnimos locais na funo objetiva a ser minimizada.
Desvantagens¶
  • Exige inverso de matriz, em um dataset muito grande aumenta muito a complexidade computacional e se torna invivel.
  • Quando no h uma margem de separao muito clara, perde a capacidade de generalizao devido a grande quantidade de rudos;

Nesse problema temos um dataset com muitas features e o algottimo SVM indicado para esse tipo de problema. Podemos testar os diferentes kernel para deixar todas essas features linearmente separveis e deixar o SVM mais eficiente.

2 - Método do gradiente estocástico (SGDC)¶

Gradiente descendente um algoritimo que busca os melhores parametros para uma funo objetivo, em outras palavras, o algotirimo encontra os valores dos coeficientes de uma funo (f) que minimiza uma funo de custo (custo).

Para encontrar os melhores coeficientes independente da inicializao o algoritimo faz diversas iteraes at encontrar os melhores coeficientes, em outras palavras, o algotirimo segue a direo da inclinao da superfcie criada pela funo objetivo em declive at chegar a um vale.

O mtodo de gradiente estocstico calcula o coeficiente que minimiza a funo objetivo para cada exemplo dos dados do treinamento, e atualiza o coeficiente.

Por padro no sklearn a implementao feita para otimizar o SVM linear, por isso faz sentido testar esse classificador para esse conjunto de dados.

Aplicação:¶

Nesse exemplo foi aplicado a descida de gradiente em uma abrodagem para criar um esquema de icentivo aos funcionrios de uma empresa.

fonte: (http://www.iosrjournals.org/iosr-jbm/papers/Vol16-issue6/Version-3/I016637780.pdf)

Vantagens¶
  • A principal diferena para o descente em lote que o estocstico atualiza o coeficiente para cada exemplo de treinamento.
  • Acaba com a redundncia executando uma atualizao por vz.
  • A grande flutuao da funo objetivo por um lado, permite que ele pule para novos e potencialmente melhores mnimos locais.
  • O aprendizado pode ser muito rpido com conjuntos muito grande de dados, e muitas vezes necessrio poucas iteraes para atingir um bom coeficiente.
Desvantagens¶
  • Realiza atualizaes frequntes fazendo com que a funo objetivo flutue fortemente.

def train_classifier(clf, X_train, y_train):
    ''' Ajusta um classificador para os dados de treinamento. '''
    
    # Inicia o relgio, treina o classificador e, ento, para o relgio
    start = time()
    clf.fit(X_train, y_train)
    end = time()
    
    # Imprime os resultados
    print "O modelo foi treinado em {:.4f} segundos".format(end - start)

    
def predict_labels(clf, features, target):
    ''' Faz uma estimativa utilizando um classificador ajustado baseado na pontuao F1. '''
    
    # Inicia o relgio, faz estimativas e, ento, o relgio para
    start = time()
    y_pred = clf.predict(features)
    end = time()
    
    # Imprime os resultados de retorno
    print "As previses foram feitas em {:.4f} segundos.".format(end - start)
    return f1_score(target.values, y_pred, pos_label='yes')


def train_predict(clf, X_train, y_train, X_test, y_test):
    ''' Treina e faz estimativas utilizando um classificador baseado na pontuao do F1. '''
    
    # Indica o tamanho do classificador e do conjunto de treinamento
    print "Treinando um {} com {} pontos de treinamento. . .".format(clf.__class__.__name__, len(X_train))
    
    # Treina o classificador
    train_classifier(clf, X_train, y_train)
    
    # Imprime os resultados das estimativas de ambos treinamento e teste
    print "Pontuao F1 para o conjunto de treino: {:.4f}.".format(predict_labels(clf, X_train, y_train))
    print "Pontuao F1 para o conjunto de teste: {:.4f}.".format(predict_labels(clf, X_test, y_test))
    print("\n")
